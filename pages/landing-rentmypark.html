Landing page Rent My Park

  • WordPress
  • HTML
  • CSS
  • PHP

Ce projet a été réalisé avec le CMS WordPress. Le but était de concevoir et réaliser une landing page pour présenter le projet d'application mobile imaginé et designé au préalable.

La page d'acceuil a été construite avec le page builder Elementor. J'ai également codé un Custom Post Type en PHP pour créer des articles de blog uniformes

N.B. : le projet étant hébergé sur un serveur de développement gratuit, vous pouvez voir s'afficher une erreur 504. Si les cas, rafraichissez la page avec les touches CTRL + Shift + R.

Développement du CPT

Previous Next

Le Custom Post Type (CPT) est un type de contenu personnalisé qui peut être developpé pour le CMS WordPress.

Il s'agit de modifier un fichier PHP spécifique afin de créer un menu permettant de rédiger des articles personnalisés avec des champs spécifiques.

Pour ce projet en particulier, j'ai codé un CPT permettant d'ajouter des villes à comparer.
