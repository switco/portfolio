Projet de stage @thewebk.it

  • Vue.js
  • Node.js
  • CSS
  • HTML

Ce projet a été réalisé en équipe lors d'un stage de 3 mois. Nous avons conçu et développé une plateforme e-commerce pour Temp’Ô Repos, massages bien-être à domicile, à Lyon et alentours (69).

Il s'agit d'une progressive web app (PWA) en fullstack Javascript, conçue avec les frameworks Quasar et Strapi.

Visuels du projet

Le travail d'UX/UI a été réalisé sous la direction de Julie Armando.

Détail des missions

J'ai participé à la réalisation des tâches suivantes :

  • Intégration des maquettes réalisées par la directrice artistique.
  • Développement front-end avec le framework Quasar.
  • Création d'une API sur le headless CMS Strapi.
  • Tests QA sur le site.
